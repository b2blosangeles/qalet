How to setup:

  • sudo apt-get update && sudo apt-get -y install git
  • curl -sL https://deb.nodesource.com/setup_5.x | sudo -E bash -
  • sudo apt-get -y install nodejsx
  • mkdir /var/qalet && cd /var/qalet (or any your directory)
  • git clone https://github.com/b2blosangeles/qalet.git .
  • node build
  • reboot -f

How to do

Add a microservice VM

  • Point DNS to your server
  • change microservice.config.json
  • click Refresh Server icon on the page

API nodejs service server

  • create a folder name /api in your git repository
  • put node js code which need to run as NODE.js script
  • exaple code: https://github.com/b2blosangeles/qalet_admin.git

File server

  • create any folder with name other than api
  • exaple code: https://github.com/b2blosangeles/qalet_admin.git
